De bestanden horen in de root map van een webserver geplaatst te worden, 
daarnaast dient de DatabaseDump te worden geinporteerd in een localhost database met kbs als database naam, 
root als user met een leeg wachtwoord
Om bij het back-end te komen moet u naar de volgende pagina navigeren: /login.php
Gebruikersnaam: admin 
Wachtwoord: P
Na het inloggen wordt u doorverwezen naar het back-end.
